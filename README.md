# Customer Behaviour Analysis

## Business Problem
The business lacked visibility into customer purchasing behaviour, spending patterns, and repeat purchases.  
This project analyses transactional data to identify customer segments, revenue drivers, and actionable insights to support better business decisions.

## Tools Used
- Python (pandas, NumPy)
- SQL (MySQL)
- Power BI

## Key Tasks Performed
- Cleaned and preprocessed raw transactional data using Python.
- Performed customer-level and product-level analysis using SQL.
- Built an interactive Power BI dashboard to visualize KPIs and business trends.
